# nodejs-express-api

API básica realizada con nodejs usando express

## Instalación
```cmd 
  git clone https://github.com/devdaniela/nodejs-express-api.git
  cd nodejs-express-api
  npm install
  node server.js
```

## Testear con Postman
```
  localhost:4000
```
